# Phase 2: Configuration (The "Anatomy")

Before you open Blender, you must tell AYON how your project is structured.

1. **Project Anatomy:** Go to **Studio Settings > Project Anatomy**.
2. **Roots:** Define where your files live.

- **Name:** `work`
- **Windows:** `P:/projects` (or whatever your local path is).
- **Linux:** `/mnt/projects`.

3. **DigitalOcean Sync:** \* Go to **Settings > Site Sync**.

- Add your DigitalOcean Spaces credentials (Key, Secret, and Endpoint) here so your 10GB animation files sync automatically.
